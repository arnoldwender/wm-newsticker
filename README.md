# WM Newsticker

A Gutenberg block for animated news tickers with scroll, fade, slide and typing animations.

## Features

- **Multiple Animations**: Scroll (marquee), fade, slide, typing
- **Dynamic Content**: Display recent WordPress posts automatically
- **Full Customization**: Colors, spacing, borders, shadows
- **User Controls**: Optional play/pause, prev/next, progress bar
- **RTL Support**: Right-to-left language support
- **Multilingual**: English, German, Spanish translations included
